## 1. 도메인 · 역할 체계

### 1.1 도메인 구조 (3개)

| 도메인 | 설명 | 포함 모듈 |
|--------|------|-----------|
| **유저 도메인** | 일반유저와 농부가 사용하는 서비스 영역 | 회원관리, 농장관리, 종자등록, 작물밸런스, AI추천, 상점, 커뮤니티, 가게정보 |
| **관리자 도메인** | 플랫폼 전체 운영·관리 영역 | 유저관리, RAG데이터관리, 가게관리, 커뮤니티관리, 상점관리, 관리자대시보드 |
| **지자체 도메인** | 지역 농업 현황 모니터링·정책 지원 영역 | 수급대시보드, 보고서, 정책알림 |

### 1.2 역할 체계 (4개)

```
┌─────────────────── 유저 도메인 ───────────────────┐
│                                                   │
│  [일반유저] ──농장 등록 신청──→ [농부]              │
│      │                           │                │
│      └─ 커뮤니티,                 └─ + 종자 등록,  │
│         상점(구매/판매),              밸런스 조회,  │
│         정책 매칭                     AI 추천,      │
│                                      가게 정보     │
└───────────────────────────────────────────────────┘

┌─── 관리자 도메인 ───┐    ┌─── 지자체 도메인 ───┐
│ [관리자]            │    │ [지자체]            │
│  전체 시스템 관리   │    │  대시보드(조회)     │
│  데이터·유저·상점   │    │  수급 보고서        │
│  RAG 관리           │    │  정책 알림 발송     │
└─────────────────────┘    └─────────────────────┘
```

## 2. 전체 메뉴 구조 (IA)

### Depth 1 → Depth 2 → Depth 3

```
FarmBalance
│
├── 🏠 홈
│   ├── 랜딩 페이지 (비로그인)
│   └── 메인 대시보드 (로그인 후, 역할별 분기)
│
├── 🔐 인증 (/auth)
│   ├── 회원가입 — 역할 선택 (일반/지자체)
│   ├── 로그인
│   └── 비밀번호 찾기
│
├── 🌾 농장 관리 (/farm)
│   ├── 농장 등록 신청
│   ├── 농장 정보 관리
│   ├── 종자 구매 등록
│   ├── 파종 계획 등록
│   └── 수확 실적 등록
│
├── 📊 밸런스 · 추천 (/balance, /recommend)
│   ├── 작물 밸런스 조회 (게이지+지도)
│   ├── 작물 밸런스 상세
│   ├── AI 작물 추천 조회
│   └── 추천 상세/근거
│
├── 🛒 상점 (/shop)
│   ├── 상품 탐색
│   ├── 상품 상세
│   ├── 장바구니
│   ├── 주문/결제
│   ├── 주문 내역
│   ├── 상품 등록/관리 (판매자)
│   └── 주문 접수/관리 (셀러)
│
├── 💬 커뮤니티 (/community)
│   ├── 게시글 목록 (카테고리별)
│   ├── 게시글 상세
│   ├── 글 작성/수정
│   └── 답변 채택
│
├── 📜 정책 매칭 (/policy)
│   └── 정책 매칭 / 공문 생성
│
├── 📍 가게 정보 (/stores)
│   ├── 가게 지도 조회 (카카오맵)
│   └── 가게 상세
│
├── 👤 마이페이지 (/mypage)
│   ├── 프로필 조회/수정 (완성도 %)
│   ├── 내 등록 이력 (종자/파종/수확)
│   ├── 포인트 내역
│   └── 알림 내역
│
├── 📈 지자체 전용 (/gov)
│   ├── 관할 지역 수급 대시보드
│   ├── 인접 지역 비교 분석
│   ├── 재배 의향 현황 (3가지 뷰)
│   ├── 수급 보고서 생성 (PDF)
│   ├── 농가 권고/공지 발송
│   └── 수급 데이터 다운로드 (CSV/XLSX)
│
└── ⚙️ 관리자 (/admin)
    ├── 관리자 대시보드
    ├── 사용자 관리
    │   ├── 사용자 목록
    │   └── 농부 승인/반려
    ├── 작물 마스터 관리
    ├── 외부 데이터 연동 관리
    ├── 밸런스 엔진 관리
    ├── 커뮤니티 관리
    ├── 상점 관리 (상품 승인/분쟁)
    ├── 가게 정보 관리
    ├── 쿠폰/보상 관리
    └── 시스템 헬스 모니터링
```

## 6. 주요 화면 흐름

### 6.1 신규 가입 → 농부 전환 흐름

```
[랜딩 페이지 /]
    │
    ├── [회원가입 /auth/signup]
    │       │
    │       ├── "일반유저" 선택
    │       │       │
    │       │       └── [가입 완료] → 즉시 이용 가능
    │       │               │
    │       │         [메인 대시보드 /mypage]
    │       │               │
    │       │         [농장 등록 신청 /farm/register]
    │       │               │
    │       │         관리자 승인 (ADM-002) 후 농부 역할 부여
    │       │               │
    │       │         [농장 관리 /farm] ← 농부 기능 활성화
    │       │
    │       └── "지자체" 선택
    │               │
    │               └── [가입 완료] → 관리자 승인 대기
    │                       │
    │               승인 후 → [지자체 대시보드 /gov]
    │
    └── [로그인 /auth/login] → [메인 대시보드]
```

### 6.2 농업 라이프사이클 흐름 (종자 → 파종 → 수확)

```
[밸런스 조회 /balance]
    │
    └── 수급 부족 작물 확인
            │
            ▼
[AI 추천 /recommend] ── Top5 추천 확인
    │
    └── [추천 상세] ── "이 작물로 시작하기" CTA
            │
            ▼
[종자 구매 등록 /farm/seed]
    │
    ├── 영수증 인증 → 100P + 50P 적립
    └── 수급 밸런스 자동 갱신 (5분 이내)
            │
            ▼
[파종 계획 등록 /farm/plan]
    │
    ├── 작물·면적·파종일 입력
    ├── 면적 초과 검증 (Soft Validation)
    └── 쏠림 120% 초과 시 알림 → AI 대안 추천 트리거
            │
            ▼
[수확 실적 등록 /farm/harvest] (Phase 2)
    │
    └── 실제 수확량·품질등급 입력 → 포인트 추가 적립
```

### 6.3 지자체 플로우

```
[지자체 대시보드 /gov]
    │
    ├── 과잉/부족 경고 배지 클릭
    │       │
    │       └── [재배 의향 현황 /gov/cultivation]
    │               ├── 작물별 뷰
    │               ├── 지역별 뷰
    │               └── 시계열 뷰
    │
    ├── [인접 지역 비교 /gov/compare] (Phase 2)
    │
    ├── [수급 보고서 /gov/reports] (Phase 2) ── PDF 생성
    │
    ├── [농가 권고 발송 /gov/notices] (Phase 2), 다채널 타겟팅
    │
    └── [데이터 다운로드 /gov/download] ── CSV/XLSX
```

### 6.4 관리자 플로우

```
[관리자 대시보드 /admin]
    │
    ├── [사용자 관리 /admin/users]
    │       └── 역할 변경, 정지, 감사 로그
    │
    ├── [농부 승인 /admin/approvals]
    │       └── 농장 등록 신청 승인/반려
    │
    ├── [작물 마스터 /admin/crops]
    │       └── CRUD + 밸런스 재계산 트리거
    │
    ├── [커뮤니티 관리 /admin/community]
    │       └── 삭제, 공지, 신고 처리
    │
    ├── [상점 관리 /admin/shop] (Phase 2)
    │       └── 상품 승인/반려/분쟁
    │
    └── [시스템 /admin/health] (Phase 2)
            └── API 상태, 히트맵
```

## 7. 역할별 메인 대시보드 구성

로그인 후 각 역할에 따라 서로 다른 대시보드를 렌더링한다.

| 역할 | 대시보드 주요 구성 |
|------|-----------------|
| **일반유저** | 밸런스 요약, AI 추천 미리보기, 커뮤니티 최신글, 농장 등록 유도 배너 |
| **농부** | 내 농장 수급 현황, 추천 작물 카드, 종자/파종 등록 현황, 포인트 잔액 |
| **지자체** | 관할 지역 수급 요약, 과잉/부족 경고, 등록률, 전년 대비 변화 |
| **관리자** | 가입 승인 대기 건수, 오늘 등록 건수, 외부 API 상태, 전체 통계 |
