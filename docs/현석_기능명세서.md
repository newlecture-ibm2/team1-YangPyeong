## 1. 역할 및 권한 정의

| 역할 코드 | 역할명 | 설명 |
|-----------|--------|------|
| `ROLE_USER` | 일반유저 | 회원가입 시 기본 역할. 커뮤니티·상점 구매·농부 등록 신청 |
| `ROLE_FARMER` | 농부 | 농장 등록 승인 후 부여. 종자 등록·판매·작물 추천 사용 |
| `ROLE_GOV` | 지자체 | 대시보드 조회 전용. 관리자가 계정 생성 |
| `ROLE_ADMIN` | 관리자 | 전체 시스템 관리 (유저·RAG·가게·커뮤니티·상점) |

## 2. 공통 기능

### 2.1 회원가입

| 항목 | 내용 |
|------|------|
| **ID** | COM-001 |
| **설명** | 이메일, 비밀번호, 이름, 연락처 입력 후 회원가입. 기본 역할 `ROLE_USER` 부여 |
| **Method** | `POST /api/auth/register` |
| **Request** | `{ email, password, name, phone }` |
| **Response** | `{ userId, email, role }` |
| **검증** | 이메일 중복 체크, 비밀번호 8자 이상 |
| **에러** | `409 DUPLICATE_EMAIL` / `400 INVALID_INPUT` |

### 2.2 로그인

| 항목 | 내용 |
|------|------|
| **ID** | COM-002 |
| **설명** | 이메일 + 비밀번호 인증 → JWT 토큰 발급 |
| **Method** | `POST /api/auth/login` |
| **Request** | `{ email, password }` |
| **Response** | `{ accessToken, refreshToken, role }` |
| **에러** | `401 INVALID_CREDENTIALS` |

### 2.3 마이페이지

| 항목 | 내용 |
|------|------|
| **ID** | COM-003 |
| **설명** | 개인정보 조회·수정, 현재 역할 확인 |
| **조회** | `GET /api/users/me` |
| **수정** | `PUT /api/users/me` → `{ name, phone }` |
| **권한** | 로그인 사용자 전체 |

## 3. 농부 전용 기능

### 3.1 농장 등록 및 관리

| 항목 | 내용 |
|------|------|
| **ID** | FRM-001 |
| **설명** | 농장 정보(이름, 주소, 토지 면적, 토양 타입) 등록. 관리자 승인 시 `ROLE_FARMER` 부여 |
| **등록** | `POST /api/farms` → `{ name, address, areaSize, soilType }` |
| **조회** | `GET /api/farms/me` |
| **수정** | `PUT /api/farms/{farmId}` |
| **상태** | `PENDING → APPROVED / REJECTED` |
| **비즈니스 규칙** | 1인 1농장. 승인 전까지 농부 기능 사용 불가 |
| **에러** | `409 FARM_ALREADY_EXISTS` / `403 NOT_APPROVED` |

### 3.2 종자 등록

| 항목 | 내용 |
|------|------|
| **ID** | FRM-002 |
| **설명** | 구매한 종자 정보 등록 |
| **등록** | `POST /api/seeds` → `{ cropName, quantity, purchaseDate, sourceStore }` |
| **조회** | `GET /api/seeds?farmId={farmId}` |
| **삭제** | `DELETE /api/seeds/{seedId}` |
| **권한** | `ROLE_FARMER` |

### 3.3 파종 계획 등록

| 항목 | 내용 |
|------|------|
| **ID** | FRM-003 |
| **설명** | 작물별 파종 계획 수립 (작물명, 파종일, 목표 재배량, 재배 면적) |
| **등록** | `POST /api/crop-plans` → `{ seedId, cropName, plantingDate, targetYield, area }` |
| **조회** | `GET /api/crop-plans?farmId={farmId}` |
| **수정** | `PUT /api/crop-plans/{planId}` |
| **삭제** | `DELETE /api/crop-plans/{planId}` |
| **비즈니스 규칙** | 재배 면적 합계 ≤ 농장 토지 면적 |
| **에러** | `400 AREA_EXCEEDED` |

### 3.4 수확 실적 등록

| 항목 | 내용 |
|------|------|
| **ID** | FRM-004 |
| **설명** | 파종 계획에 대한 실제 수확 결과 기록 |
| **등록** | `POST /api/harvests` → `{ cropPlanId, actualYield, harvestDate, qualityGrade }` |
| **조회** | `GET /api/harvests?farmId={farmId}` |
| **권한** | `ROLE_FARMER` |

### 3.5 작물 밸런스 조회

| 항목 | 내용 |
|------|------|
| **ID** | FRM-005 |
| **설명** | 양평 지역 내 작물별 재배 비율·가격·수확량 추이 시각화 |
| **재배 비율** | `GET /api/balance/crop-ratio` → 작물별 파종 면적 비율 (도넛 차트) |
| **가격 추이** | `GET /api/balance/price-trend?cropName={}&years={3}` → 연도별 가격 (라인 차트) |
| **전년 대비** | `GET /api/balance/year-comparison` → 작물별 전년 대비 증감 |
| **권한** | `ROLE_FARMER`, `ROLE_GOV`, `ROLE_ADMIN` |

### 3.6 AI 작물 추천 (RAG)

| 항목 | 내용 |
|------|------|
| **ID** | FRM-006 |
| **설명** | 토양·기후·시세 데이터 기반 AI 작물 추천 |
| **요청** | `POST /api/recommend` → `{ farmId }` |
| **응답** | `[ { cropName, score, reason, expectedRevenue } ]` |
| **RAG 참조 데이터** | 토양 정보, 기후 예측, KAMIS 가격/유통량, 지역 파종 현황 |
| **권한** | `ROLE_FARMER` |

### 3.7 가게 정보 조회

| **조회** | `GET /api/stores?lat={}&lng={}&radius={}` |
| **응답** | `[ { name, address, phone, lat, lng, category } ]` |
| **권한** | `ROLE_FARMER` |

## 10. 에러 코드 정의

| 코드 | HTTP | 설명 |
|------|------|------|
| `DUPLICATE_EMAIL` | 409 | 이미 사용 중인 이메일 |
| `INVALID_CREDENTIALS` | 401 | 로그인 정보 불일치 |
| `INVALID_INPUT` | 400 | 입력값 검증 실패 |
| `NOT_FOUND` | 404 | 리소스를 찾을 수 없음 |
| `FORBIDDEN` | 403 | 권한 없음 |
| `FARM_ALREADY_EXISTS` | 409 | 이미 농장 등록됨 |
| `NOT_APPROVED` | 403 | 농부 승인 전 접근 시도 |
| `AREA_EXCEEDED` | 400 | 재배 면적이 농장 면적 초과 |
| `STOCK_INSUFFICIENT` | 400 | 상품 재고 부족 |
| `API_CONNECTION_FAIL` | 502 | 외부 API 연동 실패 |
