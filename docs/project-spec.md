## 4. 핵심 비즈니스 로직

### 4.1 수급 계산 공식 (🟦 수경)

```
공급 예측 = Σ(등록 면적 × 작물별 단위수확량) + 외부 공급 추정
수요 예측 = 지역 인구 × 1인당 소비량 × 계절 계수
수급 비율 = 공급 예측 / 수요 예측
```

### 4.2 수급 상태 5단계

| 상태       | 수급 비율 | 설명                |
| ---------- | :-------: | ------------------- |
| 🔴 과잉경고 |  > 150%   | 즉시 대안 작물 추천 |
| 🟠 과잉주의 | 120~150%  | 주의 알림           |
| 🟢 적정     |  80~120%  | 균형 상태           |
| 🟡 부족주의 |  50~80%   | 재배 권장           |
| 🔵 부족경고 |   < 50%   | 적극 재배 권장      |

### 4.3 AI 작물 추천 엔진 파이프라인 (3단계 로직)

**1단계: 파종 시기 필터링 (사전 조건)**
* 현재 접속한 월(Month)을 기준으로 해당 시기에 파종이 불가능한 작물은 추천 후보군에서 원천 제외합니다.

**2단계: 작물 적합도 평가 (안전성 검증)**
1단계를 통과한 작물들을 대상으로 플랫폼 목적과 재배 성공률을 평가합니다.
```text
작물 적합도 점수 = (수급 적합도 × 0.50) + (환경/토양 적합도 × 0.50)
```
* **수급 적합도 (50%)**: 지역 내 작물 과잉 공급을 방지하기 위한 핵심 지표 (부족할수록 고득점)
* **환경 적합도 (50%)**: 유저 농장의 토양(pH, 유기물 등)이 해당 작물 성장에 적합한지 평가

**3단계: Top 5 추출 및 최종 정렬 (농가 훅킹 포인트)**
2단계 평가에서 **'적합도 점수'가 가장 높은 상위 5개(Top 5) 작물을 먼저 추출**한 뒤, 해당 5개 작물 내에서 **수익성(예상 수익 금액) 내림차순으로 재정렬**하여 제공합니다.
* **수익성 산출 방식**: KAMIS API의 최신 시세 전망과 농가 면적을 결합해 **"기대 예상 수익 금액(₩)"**을 도출
* **효과**: 시스템 관점에서 수급/환경 밸런스가 가장 완벽한 '최우량 5개 작물'만 추천 풀로 엄선하되, 농가에게는 직관적으로 '돈이 되는 순서'대로 노출하여 추천 시스템의 신뢰도와 파종 전환율을 동시에 잡는 최적의 UX입니다.

### 4.4 작물 등록 → 밸런스 반영 + 맞춤 분석 제공 흐름

```
농부: 작물 등록 (씨앗/종자/모종, 수량, 영수증 인증)
  → 이벤트 발행 (동시 3가지 처리)
  │
  ├─ ① 수급 반영
  │     → 수급 동적 반영 엔진 (5분 이내 갱신)
  │     → 밸런스 차트 업데이트
  │     → 파종 밀도 120% 초과 시 쏠림 알림
  │     → AI 대안 작물 추천 트리거
  │
  ├─ ② AI 정책 매칭 (등록 유도 핵심 인센티브)
  │     → 등록한 작물 + 유저 조건 기반 지자체 정책 자동 탐색
  │     → "이 작물로 받을 수 있는 지원금: 최대 ○○만원" 즉시 표시
  │     → 신청 공문 생성 바로가기 제공
  │
  └─ ③ 예측 리포트 생성 (등록 유도 부가 인센티브)
        → 예상 수확량: "○○kg (면적 × 단위수확량 기준)"
        → 예상 수익: "○○만원 (현재 시세 × 예상 수확량)"
        → 수급 전망: "현재 이 작물은 [적정/과잉/부족] 상태"
        → 주변 농가 동향: "양평군 내 ○○명이 같은 작물 재배 중"
```

### 4.5 정책 매칭 · AI 공문 생성 흐름

```
진입 경로 (2가지):
  A) 작물 등록 완료 후 → 자동 트리거 (등록 직후 결과 카드로 표시)
  B) 정책 매칭 메뉴 직접 진입 → 작물 선택 후 수동 조회

  → AI 정책 분석 엔진
      ├── 지자체 농업 지원 정책 DB 탐색 (보조금, 자재 지원, 교육 등)
      ├── 유저 조건 매칭 (지역, 면적, 작물, 경력)
      └── 매칭 결과 (적용 가능 정책 목록 + 예상 수혜 금액)
  → AI 공문 초안 생성 (Gemini API)
      ├── 정책별 신청서 양식 자동 작성
      ├── 유저 정보 자동 반영 (이름, 농장, 면적 등)
      └── PDF 다운로드 / 수정 후 제출 안내
```
