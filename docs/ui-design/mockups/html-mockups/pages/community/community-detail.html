상추 재배 시 진딧물 방제 방법 공유합니다

작성자: 김농부

상추 재배 시 진딧물이 많이 발생하는데요, 제가 사용하는 방법을 공유합니다.

1. 물리적 방제
반사 멀칭 필름을 사용하면 진딧물 유입을 줄일 수 있습니다.

2. 천적 활용
무당벌레를 방사하면 효과적입니다. 500~1,000마리/10a 권장합니다.

3. 친환경 약제
님오일(Neem oil)을 500배 희석 후 엽면 살포하면 좋습니다.
