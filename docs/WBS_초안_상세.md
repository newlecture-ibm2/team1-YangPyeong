> auth FE/BE → farm FE/BE → 카카오맵 연동 → 기상청 연동

### Phase 2: 핵심 가치 (2주차)
> balance FE/BE → KOSIS 배치 → 흙토람 연동 → 수급 밸런스 엔진

### Phase 3: AI 지능 (3주차)
> recommend FE/BE → Gemini 연동 → Bedrock 연동 → 캐싱 구현

### Phase 4: 확장 기능 (4주차)
> shop FE/BE → community FE/BE → stores FE/BE → policy FE/BE → 농사로 연동

### Phase 5: 관리/마무리 (5주차)
> mypage FE/BE → gov FE/BE → admin FE/BE → 알림 시스템

### Phase 6: 안정화 (6주차)
> 테스트 전체 → 버그 수정 → 배포 → 발표 준비
