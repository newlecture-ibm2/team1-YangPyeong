### 3.2 최대 분기점: "재배 의향 등록" vs "씨앗 구매 등록"

> ⚠️ **4개 기획안에서 가장 핵심적인 차이점**

| | 🟦 수경 | 🟩 지윤 · 🟨 지은 · 🟥 현석 |
|---|---|---|
| **데이터 수집 방식** | 농민이 **"무엇을 심을 예정"**인지 의향을 등록 | 농민이 **"무슨 씨앗을 샀는지"** 구매 정보를 등록 |
| **시점** | 파종 **전** 계획 단계 | 씨앗 **구매 후** |
| **장점** | 더 이른 시점에 수급 예측 가능 | 실제 행동 기반 → 데이터 신뢰도 높음 |
| **단점** | 의향만으로는 실행 보장 없음 | 이미 구매한 후이므로 사전 조율 여지 적음 |

> **수경만의 고유 기능**: 수급 동적 반영 엔진, 수급 상태 5단계 정의, 추천 점수 산정 공식, 시계열 추이 차트
> 
> **지윤·지은·현석 공통 고유 기능**: 직거래 상점, 커뮤니티, 카카오맵 가게 정보 — 수경에는 없음

## 10. 🏆 취합 추천안: 통합 FarmBalance 기획

> ⭐ 아래는 4명의 기획안에서 **각자의 강점을 취합**하여 만든 최적의 통합안입니다.

### 10.1 기본 정보

| 항목 | 추천안 | 출처 |
|------|--------|------|
| **제품명** | FarmBalance (팜밸런스) | 공통 |

| **슬로건** | "작은 땅에서 시작하는 큰 미래, 데이터로 균형을 맞추는 스마트 파밍" | 지윤·지은 |
| **핵심 가치** | 데이터 기반 작물 수급 예측 → 농가 소득 안정화 + 지역 농업 상생 | 수경 + 지은 |

#### 📦 유저 도메인 (일반유저 + 농부)

##### 🌱 Module 1: 작물 밸런스 시스템 (수경 기반) — 농부 전용
- 지역별 수급 현황 **지도 + 게이지 차트**

##### 🌾 Module 2: 종자 등록 & 파종 추적 (현석 + 지은 기반) — 농부 전용
- 종자 구매 등록 (종자명, 수량, 구매처, **영수증 사진 인증**)

- 추천 근거 표시 + 작물 비교 기능
- "현재 우리 동네에 양파를 심은 농가가 많습니다. 감자는 어떠신가요?" 형태의 **직관적 가이드** (지윤)

- 찜 & 단골 관리

##### 💬 Module 5: 커뮤니티 (지윤·지은·현석 공통) — 일반유저 + 농부

- 질문-답변 채택 시스템

### 10.6 MVP 범위 추천 (5주)

> **원칙**: 수경의 "핵심 루프" 아이디어 + 현석의 보수적 범위 설정을 절충

#### ✅ MVP 포함 (Phase 1: 5주)
1. 회원가입/로그인 + 역할 분리 (일반·농부·지자체·관리자)
2. 농장 등록 및 관리
3. **종자 등록 + 파종 계획** (핵심 데이터 수집 루프)
4. **작물 밸런스 시각화** (게이지 차트 + 기본 필터)
5. **AI 작물 추천** (기본 Top 5)
6. 커뮤니티 게시판 (기본 CRUD)
7. 기본 관리자 기능

#### ❌ MVP 제외 (Phase 2+)
- 직거래 상점 → Phase 2
- 카카오맵 가게 정보 → Phase 2
- 지자체 대시보드 고도화 → Phase 2
- 보상 체계 (포인트·쿠폰) → Phase 2
- 보고서 자동 생성 → Phase 3
- 수급 지도 (전국) → Phase 3
