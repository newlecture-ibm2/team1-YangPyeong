# 🌱 FarmBalance (팜밸런스) — 통합 기획안

> **작성일**: 2026-04-22
> **기반 문서**: 수경·지윤·지은·현석 4인 기획안 비교분석 취합
> **슬로건**: "작은 땅에서 시작하는 큰 미래, 데이터로 균형을 맞추는 스마트 파밍"

---

## 1. 제품 개요

### 1.1 문제 정의

소규모 농가는 **어떤 작물을 얼마나 심어야 할지** 판단할 수 있는 데이터가 부족하다.
결과적으로 특정 작물에 재배가 쏠려 가격이 폭락하거나, 반대로 공급 부족이 발생하여 농가 소득이 불안정해진다.

### 1.2 비전

**데이터 기반 작물 수급 예측**을 통해 농가 소득을 안정화하고, 지역 농업의 상생 생태계를 구축한다.

### 1.3 핵심 가치

| 가치 | 설명 |
|------|------|
| 🔄 수급 균형 | 실시간 종자 등록 데이터 → 작물 쏠림 방지 |
| 🤖 AI 추천 | 환경·시세·수급을 종합한 맞춤 작물 추천 |
| 🤝 농가 상생 | 직거래·커뮤니티·가게 정보로 농가 간 협력 |
| 📊 정책 지원 | 지자체 대시보드를 통한 데이터 기반 정책 수립 |

### 1.4 대상 지역

- **시범 지역**: 경기도 양평군
- **확대 계획**: Phase 3 이후 전국 확대

## 4. 핵심 기능 상세

### 📦 유저 도메인

#### 🌱 Module 1: 작물 밸런스 시스템 — 농부 전용

> 핵심 차별 기능. 종자 등록 데이터를 기반으로 지역 수급 현황을 실시간 시각화

- **지역별 수급 현황**: 게이지 차트 + 지도 시각화
- **수급 상태 5단계**: 과잉경고 → 과잉주의 → 적정 → 부족주의 → 부족경고
- **시계열 추이 차트**: 최근 3년 재배 트렌드
- **수급 동적 반영 엔진**: 종자 등록 시 5분 이내 수급 현황 갱신
- **수급 계산 공식**:
  - 공급 예측 = Σ(등록 면적 × 작물별 단위수확량) + 외부 공급 추정
  - 수요 예측 = 지역 인구 × 1인당 소비량 × 계절 계수

#### 🌾 Module 2: 종자 등록 & 파종 추적 — 농부 전용

> 플랫폼의 핵심 데이터 수집 루프

- **종자 구매 등록**: 종자명, 수량, 구매처, 영수증 사진 인증
- **파종 계획 등록**: 작물, 면적, 파종일, 목표 수확량
- **수확 실적 등록**: 실제 수확량, 수확일, 품질 등급
- **데이터 흐름**: 종자 등록 → 실시간 파종 밀도 계산 → 쏠림 시 알림

#### 🤖 Module 3: AI 작물 추천 — 농부 전용

- **추천 점수 산정 공식**:
  - 수급 적합도 35% + 환경 적합도 25% + 수익성 25% + 농가 역량 15%
- **AI 엔진**: Gemini API 기반 프롬프트 추천
- **UX**: "현재 우리 동네에 양파를 심은 농가가 많습니다. 감자는 어떠신가요?" 형태의 직관적 가이드
- 추천 근거 표시 + 작물 간 비교 기능
- Top 5 추천 작물 리스트

#### 🛒 Module 4: 직거래 셀러 허브 — 일반유저(구매) + 농부(판매)

- 스마트폰 사진 한 장으로 간편 상품 등록 (농부)
- 소규모 셀러용 주문/결제 대시보드
- 장바구니 + 찜 & 단골 관리

#### 💬 Module 5: 커뮤니티 — 일반유저 + 농부

- 카테고리별 게시판 (자유, 정보공유, Q&A)
- 질문-답변 채택 시스템
- 기본 CRUD + 검색

#### 🗺️ Module 6: 가게 정보 — 농부 전용

- 카카오맵 API 연동 — 인근 농자재 가게 검색
- "초보 친화도" 리뷰 시스템
- 제휴 가게 상단 노출 + 공동구매 매칭

## 5. 유저 플로우

### 5.1 신규 농부 온보딩

```
회원가입 → 일반유저 → 농장 등록 신청 → 관리자 승인 → 농부 역할 부여
                                                        ↓
                                              종자 등록 → 밸런스 조회 → AI 추천
```

### 5.2 핵심 데이터 수집 루프

```
농부가 종자 구매 → 플랫폼에 종자 등록 (영수증 인증)
        ↓
수급 동적 반영 엔진 (5분 이내 갱신)
        ↓
작물 밸런스 차트 업데이트
        ↓
쏠림 감지 시 → 농부 알림 + 지자체 경고
        ↓
AI가 대안 작물 추천
```

### 5.3 직거래 흐름

```
농부: 상품 등록 (사진 + 설명 + 가격)
        ↓
일반유저/농부: 상품 탐색 → 장바구니 → 주문
        ↓
농부: 주문 확인 → 발송
```
