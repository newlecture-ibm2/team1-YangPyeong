# 📑 FarmBalance — 기능명세서 (Functional Specification)

| 항목 | 내용 |
|------|------|
| **프로젝트명** | FarmBalance (팜밸런스) |
| **버전** | v1.0 (MVP) |
| **작성일** | 2026-04-21 |

---

## 1. 사용자 역할 정의

| 역할 코드 | 역할명 | 설명 | 접근 권한 |
|-----------|--------|------|-----------|
| `FARMER` | 농민 | 씨앗 등록, 작물 추천, 직거래, 커뮤니티 | 농민용 서비스 전체 |
| `ADMIN` | 관리자(지자체) | 모니터링, 권고 발송, 가게 관리 | 관리자 대시보드 전체 |
| `STORE` | 종자 판매점 | 가게 정보 등록, 공동구매 참여 | 가게 관리 페이지 |
| `GUEST` | 비회원 | 작물 현황 조회, 커뮤니티 열람 | 읽기 전용 |

### 2.3 AI 작물 추천 (RECOMMEND)

| ID | 기능명 | 설명 | 우선순위 |
|----|--------|------|----------|
| REC-01 | 토양 분석 조회 | 흙토람 API 연동, 내 토지의 토양 특성 표시 | P0 |
| REC-02 | 작물 점수 산출 | 토양 + 기후 + 수급 데이터 종합 작물별 점수 | P0 |
| REC-03 | 대체 작물 추천 | 과잉 작물 감지 시 대체 작물 3개 추천 | P0 |
| REC-04 | 수급 최적화 알림 | "양파 과다 → 감자 전환 시 수익 n%↑" 푸시 알림 | P1 |
| REC-05 | AI 상세 분석 | Gemini API 기반 맞춤형 재배 가이드 | P2 |

**API 명세:**

| Method | Endpoint | 설명 | Request |
|--------|----------|------|---------|
| GET | `/api/recommend/soil?lat={}&lng={}` | 토양 정보 조회 | 위경도 |
| POST | `/api/recommend/score` | 작물 점수 산출 | `{landArea, soilType, region}` |
| GET | `/api/recommend/alternatives?crop={cropId}` | 대체 작물 추천 | 현재 작물 |
| POST | `/api/recommend/ai-guide` | AI 상세 분석 요청 | `{landArea, soilType, budget}` |

**AI 프롬프트 구조:**
```
Context: 토지 {landArea}평, 경기도 양평 {region}, {soilType}, 
         주변 {crop} 파종량 {percentage}% 초과
Task:    리스크 최소화 + {budget}만원 이상 수익 가능 대체 작물 3가지 추천
Output:  작물명, 예상 수익, 리스크 점수(1~10), 추천 사유
```

## 4. 데이터 모델 (ERD 개요)

### 4.1 주요 엔티티

| 엔티티 | 필드 | 설명 |
|--------|------|------|
| **User** | id, name, phone, password, role, region, landArea, points, createdAt | 사용자 |
| **Crop** | id, name, category, avgPrice, growthPeriod | 작물 마스터 |
| **SeedRegistration** | id, userId, cropId, quantity, sowingDate, receiptImageUrl, verified, createdAt | 씨앗 등록 |
| **Product** | id, sellerId, name, price, stock, description, imageUrl, status, createdAt | 직거래 상품 |
| **Order** | id, buyerId, totalPrice, status, createdAt | 주문 |
| **OrderItem** | id, orderId, productId, quantity, price | 주문 항목 |
| **CartItem** | id, userId, productId, quantity | 장바구니 |
| **Post** | id, authorId, title, content, category, viewCount, createdAt | 게시글 |
| **Comment** | id, postId, authorId, content, accepted, createdAt | 댓글 |
| **Store** | id, name, address, lat, lng, phone, partnerStatus, createdAt | 가게 |
| **Coupon** | id, storeId, discount, validUntil, usedBy | 쿠폰 |
| **GuideMessage** | id, adminId, cropId, content, sentAt | 권고 메시지 |
| **PointHistory** | id, userId, amount, reason, createdAt | 포인트 이력 |

### 4.2 엔티티 관계

```
User (1) ──── (N) SeedRegistration ──── (1) Crop
User (1) ──── (N) Product
User (1) ──── (N) Order ──── (N) OrderItem ──── (1) Product
User (1) ──── (N) CartItem ──── (1) Product
User (1) ──── (N) Post ──── (N) Comment
User (1) ──── (N) PointHistory
Store (1) ──── (N) Coupon
Admin (1) ──── (N) GuideMessage ──── (1) Crop
```
